## Text Commands

You can also type commands in the chat interface:

1. Type your command in the input field
2. Press Enter or click Send
3. Jarvis will respond and execute actions

## Quick Actions

Click the quick action buttons on the right panel:
- 🌐 **Chrome** - Launch Chrome browser
- 🦊 **Firefox** - Launch Firefox browser
- 💻 **VS Code** - Launch Visual Studio Code
- 🧮 **Calculator** - Launch Calculator

## Features

### Real-Time Camera Feed
- View live camera feed in the left panel
- See hand detection overlay
- Real-time hand tracking

### 3D Model Viewer
- Interactive 3D model display
- Rotating models
- Multiple model types (sphere, cube, pyramid, robotic hand)

### Voice Control
- Click "Start Listening" button
- Speak your command clearly
- Jarvis will transcribe and execute

### Chat Interface
- Type or speak commands
- See response from Jarvis
- Access command history
- View system status

## Tips & Tricks

### For Better Voice Recognition
1. Speak clearly and naturally
2. Use complete sentences
3. Minimize background noise
4. Position microphone properly

### For Better Hand Detection
1. Ensure good lighting
2. Keep hands clearly visible
3. Adjust camera position
4. Clear hand gestures

### For Better AI Responses
1. Be specific in your commands
2. Use natural language
3. Provide context when needed
4. Ask follow-up questions

## Keyboard Shortcuts

- **Enter** - Send text command
- **Ctrl+Shift+M** - Start/stop microphone (coming soon)
- **Ctrl+Shift+H** - Show/hide hand detection (coming soon)

## System Status

The right panel shows:
- **Voice Enabled** - Voice recognition status
- **Hand Detection** - Hand tracking status
- **AI Brain** - Backend connection status

## Troubleshooting

### Commands Not Recognized
- Check microphone is working
- Speak louder and clearer
- Check background noise
- Restart listening

### 3D Models Not Showing
- Ensure WebGL is enabled
- Check browser compatibility
- Reload the page
- Check browser console for errors

### Apps Not Launching
- Verify app is installed
- Check app path in config
- Try launching manually first
- Check system permissions
